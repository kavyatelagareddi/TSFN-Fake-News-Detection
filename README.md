# TSFN-Based Early Fake News Detection Framework

## Overview
A hybrid fake news detection framework using SBERT embeddings, XGBoost, and social propagation analysis for early misinformation detection.

## Features
- SBERT semantic embeddings
- Graph-based propagation analysis
- TSFN feature fusion
- XGBoost classification
- Top-K early spreader analysis

## Technologies Used
- Python
- SBERT
- XGBoost
- Scikit-learn
- NetworkX
- Pandas
- NumPy
